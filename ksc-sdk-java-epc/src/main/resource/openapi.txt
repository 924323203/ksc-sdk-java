1.4.65. CreateCabinet (创建机柜)
1.4.65.1. Request Parameters(请求参数)
CabinetName

云物理主机托管机柜名称，默认为Cabinet
类型：String
是否必填：否
缺省值：Cabinet
格式：2-128个字符，支持中英文数字特殊符号
ChargeType

主机计费类型，按日月结Daily
有效值：
PrePaidByMonth ：包年包月，有到期时间
PostPaidByDay：按日月结，无到期时间
类型： String
是否必填：是
CabinetType

云物理主机托管机柜类型，托管机柜Normal，预留机柜Reserved
类型：String
有效值：
Normal：标准托管租赁
Reserved：预留机柜
AvailabilityZone

可用区的名称
类型:String
是否可缺省: 是
PurchaseTime

购买时长
类型：Integer
可取值：1-36
是否必填：否
1.4.65.2. Response Elements(返回值)
RequestId

请求ID
类型：String
CabinetId

托管机柜ID
类型：String

1.4.66. DescribeCabinets(获取云物理主机托管机柜列表信息)
1.4.66.1. Request Parameters(请求参数)
CabinetId.N

托管机柜ID，如果有ID则查询具体机柜，如果没有ID则查询机柜列表信息
类型：String
是否必填：否
MaxResults

单次调用可返回的最大条目数量. 传入返回的 NextToken 值可以获取剩余的其它条目. 这个值可以允许的范围是 5- 1000.
类型: Integer
是否必填：否
NextToken

获取另一页返回结果的 token.
类型: String
是否必填：否
1.4.66.2. Response Elements(返回值)
RequestId

请求ID
类型：String
CabinetSet

托管机柜资源列表
类型：Cabinet List
TotalCount

托管机柜总数
类型：Int
